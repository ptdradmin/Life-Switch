# 🤝 Bienvenue sur Life Switch !

Merci de t'intéresser au projet. Notre mission est de fournir un outil de confiance pour la succession numérique. En tant que projet Open Source, nous valorisons les contributions qui améliorent la sécurité, la clarté du code et l'expérience utilisateur.

## 🛠️ Comment contribuer ?

1.  **Signaler un bug** : Ouvre une issue en décrivant le problème et les étapes pour le reproduire.
2.  **Proposer une amélioration** : Que ce soit une nouvelle fonctionnalité pour Carnet AI ou une optimisation de l'interface.
3.  **Soumettre du code** :
    *   Fork le dépôt.
    *   Crée une branche descriptive (ex: `feature/nouvelle-langue` ou `fix/bug-notif`).
    *   Assure-toi que ton code respecte les standards (TypeScript, Tailwind).
    *   Soumets une Pull Request (PR) détaillée.

## ⚖️ Code de conduite

-   Sois respectueux et bienveillant envers les autres contributeurs.
-   Privilégie la sécurité : si tu découvres une vulnérabilité critique, merci de nous contacter en privé avant de la rendre publique.

## 🌍 Internationalisation (i18n)

Si tu souhaites ajouter une langue ou corriger une traduction :
-   Modifie le fichier `src/i18n/translations.ts`.
-   N'oublie pas d'ajouter le nouveau flag dans `src/pages/SettingsPage.tsx`.
